# hotel-mangment-system
A Java-based desktop application built with OOP and JDBC to streamline hotel operations, featuring role-based authentication, room tracking, and guest management.
# How to Run the Project
1. **Download the code:** Click on the green `<> Code` button at the top and select **Download ZIP**, or clone the repository.
2. **Open the project:** Extract the downloaded ZIP file. Since the project includes a `build.xml` file, you can easily open it directly as a project in **NetBeans IDE** (or any IDE that supports Apache Ant).
3. **Database Setup:** 
   * Ensure you have MySQL installed.
   * Create the required database.
   * Update the JDBC connection string, username, and password in the database connection class to match your local setup.
4. **Run:** Execute the Main class to launch the Hotel Management interface!
